El proyecto contiene dos botones iniciales, uno que te dirige a los estudiantes y el otro a las carreas, primero se debe crear al menos una carrera ya que al querer ingresar un nuevo estudiante el formulario no deja enviarlo si no hay una carrera seleccionada, así que en caso de tener la base de datos aun vacia de la parte de la carrera primero se ingresa una carrera almenos y ya se puede registrar un alumno, dentro de cada apartado como es el apartado de estudiantes donde se puede ver cada alumno en lista, en la fila de cada dato estan los botones de editar y el boton de eliminar, los botones para agregar estan arriba a la derecha, esto es igual para el apartado de las carreras.
